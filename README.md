### 🚀 Sobre mí

Construyo productos **end-to-end**: desde la app que usa el cajero hasta el backend, la base de datos y el dashboard de administración.

- 🏪 **POS / Retail:** facturación electrónica (ARCA/AFIP), control de caja y cierre Z, gestión de stock, impresión en impresoras térmicas y lectores de código de barras.
- 🤖 **Automatización (RPA):** bots en Python + Selenium que sincronizan stock y precios entre sistemas ERP y tiendas WooCommerce, con auditoría y reportes automáticos.
- 🧠 **Integración de IA:** chatbots con RAG (OpenAI) y asistentes ("coach") dentro de apps móviles.
